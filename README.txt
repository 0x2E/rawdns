
        rawdns
        ------


        A crude implementation of a DNS messages and raw UDP queries
        for fetching A records from recursive DNS servers.


        Usage: rawdns [--address ADDRESS] HOSTNAME

        Positional arguments:
          HOSTNAME               name to resolve

        Options:
          --address ADDRESS [default: 8.8.8.8:53]
          --help, -h             display this help and exit
